## Ключевые изменения

1. **RF-001:** Введение Data Mapper — разделение Entity и DTO
2. **RF-002:** Замена `.get()` на `.orElseThrow()`
3. **RF-003:** Вынесение JWT-секрета в конфигурацию
4. **RF-004:** Единый GlobalExceptionHandler
5. **RF-005:** Оптимизация N+1 запросов в CartRepository
6. **RF-006:** Рефакторинг Android ViewModel
7. **RF-007:** Валидация количества в корзине
8. **RF-008:** Индексация часто используемых запросов
