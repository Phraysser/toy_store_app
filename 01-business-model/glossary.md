# Бизнес-глоссарий проекта «Toy Store»

| Термин | Определение | Синонимы |
|--------|-------------|----------|
| **Игрушка** | Товар в каталоге магазина с характеристиками: название, описание, цена, категория, изображение, остаток на складе. | Product, Item, Товар |
| **Корзина** | Временный контейнер, содержащий выбранные пользователем товары для покупки с указанием количества. | Shopping Cart, Cart |
| **Категория** | Группа товаров для организации каталога (например: мягкие игрушки, конструкторы, образовательные игрушки). | Category, Раздел |
| **Заказ** | Оформленная покупка, содержащая список товаров, общую сумму, статус оплаты и доставки. | Order, Purchase |
| **Пользователь** | Зарегистрированный клиент магазина, имеющий личный кабинет и историю заказов. | User, Customer, Покупатель |
| **JWT токен** | JSON Web Token, используемый для безопасной аутентификации пользователя при запросах к серверному API. | Access Token, Bearer Token |
| **Роль** | Набор прав доступа в системе. Выделяются роли «Пользователь» (базовые права) и «Администратор» (управление каталогом). | Role, Права доступа |
| **Оффлайн-режим** | Возможность просмотра каталога и работы с корзиной при отсутствии подключения к сети с последующей синхронизацией. | Offline Mode |
| **Серверная часть** | Программный компонент (Spring Boot), обеспечивающий хранение данных, бизнес-логику и REST API для мобильного клиента. | Backend, Сервер |
| **Клиентская часть** | Мобильное приложение на Android (Kotlin + Jetpack Compose), отвечающее за отображение данных и взаимодействие с пользователем. | Frontend, Клиент, Mobile App |
| **Push-уведомление** | Уведомление, отправляемое пользователю на мобильное устройство о статусе заказа или специальных предложениях. | Notification, Alert |
| **Поиск** | Функция быстрого нахождения игрушки по названию, описанию или категории. | Search, Фильтр |
| **Избранное** | Список товаров, добавленных пользователем в избранное для быстрого доступа. | Wishlist, Favorites |
