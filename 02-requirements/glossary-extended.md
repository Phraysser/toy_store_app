# Расширенный глоссарий проекта «Toy Store»

| Термин | Определение |
|--------|-------------|
| **Пользователь** | Зарегистрированное лицо, использующее мобильное приложение для покупки игрушек и управления заказами. |
| **Игрушка (Товар)** | Единица каталога с характеристиками: название, описание, цена, категория, изображение, остаток на складе. |
| **Категория** | Группа товаров для организации каталога (например: мягкие игрушки, конструкторы, образовательные игрушки). |
| **Корзина** | Временный контейнер, содержащий выбранные пользователем товары для покупки с указанием количества. |
| **Заказ** | Оформленная покупка, содержащая список товаров, общую сумму, статус оплаты и доставки. |
| **Напоминание** | Push-уведомление, отправляемое пользователю о статусе заказа или специальных предложениях. |
| **Статус заказа** | Состояние заказа: «Создан» (CREATED), «Оплачен» (PAID), «Отправлен» (SHIPPED), «Доставлен» (DELIVERED), «Отменён» (CANCELLED). |
| **Синхронизация** | Процесс обмена данными между мобильным клиентом и сервером для обеспечения актуальности каталога и заказов. |
| **Оффлайн-режим** | Возможность работы с приложением без подключения к сети, с сохранением изменений в локальной БД (Room). |
| **JWT токен** | JSON Web Token, используемый для безопасной аутентификации пользователя при запросах к серверному API. |
| **Роль** | Набор прав доступа в системе. Выделяются роли «Пользователь» (USER) и «Администратор» (ADMIN). |
| **Поиск** | Функция быстрого нахождения игрушки по названию, описанию или категории. |
| **Фильтрация** | Возможность отображения товаров по определённой категории или ценовому диапазону. |
| **Серверная часть** | Программный компонент (Spring Boot), обеспечивающий хранение данных, бизнес-логику и REST API для мобильного клиента. |
| **Клиентская часть** | Мобильное приложение на Android (Kotlin + Jetpack Compose), отвечающее за отображение данных и взаимодействие с пользователем. |
| **BCrypt** | Алгоритм хеширования паролей, устойчивый к атакам перебором. |
| **REST API** | Архитектурный стиль взаимодействия компонентов распределённого приложения посредством HTTP-запросов. |
| **OpenAPI (Swagger)** | Спецификация для описания REST API, включающая интерактивную документацию и возможность тестирования. |
| **Spring Boot** | Фреймворк для создания автономных Java-приложений с минимальной конфигурацией. |
| **PostgreSQL** | Объектно-реляционная система управления базами данных с полной поддержкой ACID. |
| **PCMEF** | Архитектурный паттерн (Presentation-Control-Mediator-Entity-Foundation) для клиент-серверных приложений. |
| **Модульное тестирование** | Тестирование отдельных компонентов программного кода (классов, методов) в изоляции от внешних зависимостей. |
| **JaCoCo** | Инструмент для измерения покрытия кода тестами в Java-проектах. |
| **Локальное кэширование** | Хранение данных на устройстве пользователя (Room Database) для обеспечения работы приложения без сети. |
| **DTO** | Data Transfer Object — объект для передачи данных между подсистемами (клиент ↔ сервер). |
| **Entity** | Класс, представляющий строку таблицы базы данных в рамках JPA/Hibernate. |
| **Repository** | Паттерн, абстрагирующий доступ к хранилищу данных. |
| **Service** | Слой бизнес-логики в архитектуре Spring (аннотация @Service). |
| **Controller** | Компонент, обрабатывающий входящие HTTP-запросы в Spring MVC (@RestController). |
| **CORS** | Cross-Origin Resource Sharing — механизм, разрешающий веб-страницам запрашивать ресурсы с другого домена. |
| **DDL** | Data Definition Language — подмножество SQL для определения структуры базы данных (CREATE, ALTER, DROP). |
| **ViewModel** | Компонент Android Architecture Components, хранящий UI-данные и переживающий поворот экрана. |
| **StateFlow** | Observable-контейнер данных из Kotlin Coroutines, автоматически обновляющий UI при изменении состояния. |
| **Retrofit2** | Библиотека для типобезопасной работы с HTTP API в Android-приложениях. |
| **Hilt** | Библиотека для внедрения зависимостей (Dependency Injection) в Android-приложениях. |
| **Jetpack Compose** | Современный инструментарий для создания нативных UI в Android-приложениях на Kotlin. |
